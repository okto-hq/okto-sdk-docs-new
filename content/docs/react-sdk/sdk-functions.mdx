The Okto React SDK provides a comprehensive set of functions to manage user authentication, portfolio data, supported networks and tokens, user details, wallet management, order history, NFT order details, transaction statuses, token and NFT transfers, raw transaction executions, and UI theme settings. 

Below is an overview of the available functions and their respective purposes.

## Authentication Functions
| Function | Description |
|----------|-------------|
| [`authenticate(idToken, callback)`](/docs/react-sdk/authenticate-users/google-oauth/auth-user-via-code#authenticate-a-user) | Authenticates user with Google OAuth |
| [`sendEmailOTP(email)`](/docs/react-sdk/authenticate-users/email-otp/auth-user-via-email#send-email-otp) | Sends OTP to user's email |
| [`verifyEmailOTP(email, otp, token)`](/docs/react-sdk/authenticate-users/email-otp/auth-user-via-email#verify-email-otp) | Verifies email OTP |
| [`sendPhoneOTP(phoneNumber, countryShortName)`](/docs/react-sdk/authenticate-users/phone-otp/auth-user-via-phone#send-phone-otp) | Sends OTP to user's phone |
| [`verifyPhoneOTP(phoneNumber, countryShortName, otp, token)`](/docs/react-sdk/authenticate-users/phone-otp/auth-user-via-phone#verify-phone-otp) | Verifies phone OTP |
| [`showOnboardingModal()`](/docs/react-sdk/authenticate-users/onboarding-modal/auth-user-via-modal#show-onboarding-modal) | Presents a guided onboarding flow to the user |

|----------|-------------|

| [`getAuthDetails()`](/docs/react-sdk/authenticate-users/login-logout/fetch-auth-details#fetch-authentication-details-of-the-current-user) | Fetches the authentication details of the currently logged-in user |
